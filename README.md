## keybinds
* wasd, space, ctrl, shift: move
* esc: quit
* b: capture mouse	
* e: render mode line
* c: 2d camera
* f: player physics
* tab: debug text visible
* mousewheel: change selected block

## dependencies
* SDL3, 
* sonst alles mit drin
